RepBot

AI-Powered Fitness Tracking Application

About RepBot

RepBot is a Flutter-based fitness tracking application that uses AI-powered pose detection to automatically count exercise repetitions in real-time using your device’s camera. It combines computer vision, local data tracking, and AI analysis to give users a complete fitness experience — from form practice to personalized recommendations.

Features

Core

  • Push-up Counter: Detects push-up reps using shoulder, collarbone, and ankle positions.
  • Pull-up Counter: Tracks pull-ups by measuring hand–body center distance.
  • Squat Counter: Uses knee angles to detect correct squat form.
  • Dumbbell Curl Counter: Detects elbow angle changes for bicep curls.
  • Free Play Mode: Skeleton overlay without rep counting for practicing form.

Progress Tracking

  • Daily Tracker: Shows today’s total reps per exercise in a simple report.
  • Streak Tracking: Tracks consecutive days of workouts and resets if missed.

AI-Powered

  • AI Fitness Analyzer: Collects user data and workout logs for personalized insights.
  • Tutorial Slides: Step-by-step guide on camera placement and exercise setup.

Technical Details

Pose Detection

  • Built on Google ML Kit’s pose detection for real-time skeleton tracking.
  • Supports major landmarks: shoulders, elbows, wrists, hips, knees, ankles.
  • Handles camera frame rotations and format conversions.

Repetition Counting

  • Push-ups: Down → Up transitions with cooldown.
  • Pull-ups: Distance thresholds vs. shoulder–hip baseline.
  • Squats: Knee angles (below 100° = down, above 160° = up).
  • Curls: Elbow angles (below 70° = up, above 150° = down).

Local Data Storage

  • SharedPreferences for exercise logs & streak data.

Tech Stack

  • Frontend: Flutter
  • ML & Vision: Google ML Kit Pose Detection
  • Persistence: SharedPreferences
  • Backend (AI Analysis): REST API

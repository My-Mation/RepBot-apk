RepBot

AI-Powered Fitness Tracking Application

About RepBot

RepBot is a Flutter app that uses AI-powered pose detection to automatically count exercise repetitions in real-time via the device's camera. Track your workouts effortlessly with advanced machine learning technology.

Core Features

Exercise Counters

  • Push-up Counter: Tracks push-ups by detecting shoulder and collarbone movement. Counts reps with a 1-second cooldown.
  • Pull-up Counter: Monitors pull-ups by calculating distance between body center and hands.
  • Squat Counter: Uses knee angle calculations to detect squats.
  • Dumbbell Curl Counter: Tracks bicep curls by measuring elbow angles.

Each counter includes live rep count, camera preview with skeleton overlay, status text, FPS, and diagnostics.

Free Play Mode

A practice mode with camera preview and full-body skeleton overlay, perfect for form practice without tracking.

Daily Tracker

Displays today's total reps for each exercise in a dialog. Updates dynamically after workouts.

Streak Tracking

Calculates and displays your current exercise streak. Shows historical logs with details.

AI Fitness Analyzer

Collects user data and sends to a backend API for personalized AI analysis and feedback.

Tutorial

Interactive slideshow with instructions for each exercise and AI analysis.

Technical Features

  • Google ML Kit Pose Detection for real-time skeleton tracking
  • Local data logging and storage using SharedPreferences
  • Front camera processing with high resolution
  • Dark theme UI with responsive design
  • Wakelock to prevent screen timeout
  • Debug mode and error handling
